The chatbot supports the following keywords:
  • INFO - to display information and list of commands.
  • CASES - to display total cases.
  • ACTIVE new on v4 - to display current active cases
  • DEATHS - to display total deaths.
  • RECOVERED - to display total recoveries.
  • CRITICAL new on v4 - to display critical cases.
  • TESTS new on v4 - to display number of tests conducted.
  • RATES - display recovery and fatality rate.
  • TODAY - display new cases for the day.
  • TOTAL new on v4 - display the total cases including active, deaths, and recoveries.
  • PER MILLION new on v4 - to display statistics per one million.
